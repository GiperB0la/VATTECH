В директории <bin> находится исполняемый файл <Vattech.exe>.
В директории <Folders> находятся папки треков.

Так же можно собрать проект через cmake, прописав:
> mkdir build
> cd build
> cmake ..
> cmake --build .

Ещё можно передать путь к папке с треками через командную строку:
> Vattech.exe "Path\Folders"
